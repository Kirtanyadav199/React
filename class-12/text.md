type of routers:-

1) Browser Router :
      - Uses History API
      - Clean URL 
      - used in modern apps
      - Good SEO
      - Needs server configuration 
      - used in client side rendering 


2) Hash Router :
      - contains /courses/#/about
      - ugly urls
      - No SEO
      - used in Older Browsers 


3) Memory Router :
    - React native 

4) Static router    
    - used in server side rendering
